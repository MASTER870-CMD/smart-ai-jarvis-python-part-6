## 🌌 Overview

AI NEXUS is a futuristic desktop AI interface built using:

- ⚡ Flask (Backend Server)  
- 🧠 Python (Core Logic)  
- 🖥️ PyWebView (Desktop UI)  
<img width="1365" height="767" alt="Screenshot 2026-03-20 204713" src="https://github.com/user-attachments/assets/03b6008b-e158-4d3a-9d71-2a84496a9467" />


It transforms a web-based AI system into a **real desktop application** with a Neural Interface feel — inspired by JARVIS.

## ✨ Key Features

### 🧠 Neural Interface UI
- Fully immersive desktop experience  
- Runs like a native application  
- Smooth window rendering using WebView

### ⚙️ Background Flask Engine
- Runs Flask server in a separate thread  
- Handles backend AI operations  
- Fast & efficient multi-threading  

### 🖥️ Desktop Integration
- Converts web app → desktop app  
- Custom window size & controls  
- Seamless UI interaction
